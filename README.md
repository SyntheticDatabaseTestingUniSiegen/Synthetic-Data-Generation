# Synthetic Data Generator - PQS

AI-powered web app to generate SQL schemas and realistic synthetic data for testing databases.

## Features

- Generate SQL schema from prompts or custom input  
- Create realistic synthetic data for multiple tables  
- Select pivot rows and generate SQL queries  
- Full frontend implementation (HTML, CSS, JS)  

## How to Use

1. Open `index.html` in a browser.  
2. Select input type: **Prompt → Schema → Data**, **Schema → Data**, or **CSV Upload**.  
3. Enter your OpenRouter API key.  
4. Generate schema and data.  
5. View pivot rows and generate queries.  

## Tech Stack

- HTML / CSS / JavaScript  
- OpenRouter AI for schema and data generation  

## Notes

- Make sure to have a valid OpenRouter API key.  
- For local testing, open `index.html` in your browser.
